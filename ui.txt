1. ui_core.c —— UI 核心
   - 初始化整个 UI 系统（LCD 初始化、参数限制初始化、波形模式初始化）
   - 运行 UI 主任务循环 `ui_task()`，从消息队列获取按键事件并分发给 `keyval_process()`
   - 维护全局状态：`selected_index`（当前选中菜单项）、`selected_ch`（当前选中通道）、`signal_mode`（当前工作模式）
   - 在无按键时自动刷新测量数据

2. ui_display.c —— 显示管理
   - 绘制上方/下方通道的圆角矩形参数区域（通道1=黄色，通道2=青色）
   - 显示通道参数标签：频率、幅度、偏置、占空、相位
   - 显示通道使能状态（ON/OFF，绿色/红色）
   - 匹配下方通道的波形类型名称和未选中通道的参数数值

3. ui_draw.c —— 基础绘制
   - 提供屏幕布局的基础绘制函数：
     · DrawLeftBigRect()     — 左侧大圆角矩形边框（主菜单区域）
     · DrawLeftTopRect()     — 左上角小矩形（通道标签区）
     · DrawLeftTopRect_2()   — 左下角小矩形（第二通道标签区）
     · DrawTopMiddleRect()   — 顶部中间小矩形（模式标题区）
   - 所有矩形内填充黑色以形成"空心"边框效果

4. ui_key.c —— 按键处理核心
   - `keyval_process()` 是整个系统的按键事件分发中枢
   - 处理多种按键：正弦波/矩形波/任意波快捷键、OK键、左右方向键、CH1/CH2切换键、EC11旋钮（正转/反转）
   - 根据当前模式路由到不同处理逻辑：
     · MODE_WAVE  → 数字位选择、数值增减、波形类型切换
     · MODE_SWEP  → 扫描参数选择与调节
     · MODE_MEAS  → 测量菜单操作
     · MODE_SYNC  → 同步菜单设置
     · MODE_SAVE/LOAD → 存储/加载菜单
     · MODE_SYS   → 系统菜单操作
   - EC11旋钮调节参数后自动触发 `sync_copy()`（同步复制）、`limit_amp_offset()`（幅度偏置限制）、波形刷新和 DAC 输出更新
   - 特殊波形（AM调幅/FM调频/CHIRP线性调频）在参数变更时自动启动扫描

5. ui_layout.c —— 界面布局
   - 定义网格线的 Y 坐标常量（10条水平分割线 GRID_Y_0 ~ GRID_Y_9）
   - `DrawDashe_tips()` 绘制虚线的界面网格布局（水平分割线 + 垂直分隔线）
   - 将 320×240 屏幕划分为参数显示区、波形预览区、菜单区

6. ui_limit.c —— 参数限制管理
   - 初始化所有参数的 NumericController（数字控制器），设定取值范围和位数：
     · 频率: 1~100000 Hz, 6位
     · 幅度: 1~3300 mV, 4位
     · 偏置: 1~3300 mV, 4位
     · 占空: 1~99%, 2位
     · 相位: 0~360°, 3位
   - `limit_amp_offset()` 实现幅度与偏置的互相约束：幅度+偏置 ≤ 3300mV，偏置 ≥ 幅度
   - `Sweep_Limit()` 根据扫描参数类型（频率/幅度/偏置/占空）设置不同的起止范围
   - 管理扫描参数的数字控制器（NUB_BEGIN起始值 / NUB_END截止值 / NUB_TIME时间）

7. ui_measure.c —— 测量模式界面
   - 显示测量数据：频率/计数、周期(us)、正脉宽(us)、负脉宽(us)、占空比(%)
   - 闸门时间设置：1S / 10S / 100S 三档可选
   - `freq_or_count()` 切换频率/计数显示模式
   - `pause_or_start()` 暂停/继续测量
   - `measure_clear()` 清零测量数据
   - `measure_mode_init()` 初始化整个测量界面布局
   - `refresh_measure_data()` 持续刷新显示所有测量值

8. ui_menu.c —— 菜单系统
   - 定义菜单项中文字符串数组 `str_chinese[]`（频率/幅度/偏置/占空/相位/对象/截止/时间/模式/方向/存储/加载/同步/配置/更多/计数/暂停/清零/闸门/波形）
   - 定义 5 种菜单布局 `menu_str[][]`：波形菜单、扫描菜单、系统菜单、测量菜单、同步菜单
   - `DrawMenuSelected()` 在右侧菜单栏绘制选中项的圆角矩形高亮效果（外框+内黑色边框）
   - 提供各模式的菜单设置函数：`wave_menu_set()` / `sweep_set()` / `system_set()` / `measure_set()` / `sync_menu_set()`
   - 定义 30+ 种波形类型的中文名称数组 `wave_type_str[]`（正弦波/矩形波/三角波/锯齿波/指数波/对数波/噪声波/ECG/AM/FM/CHIRP等）
   - `DrawWaveformSelect()` 在通道标签区显示当前波形类型名称并支持任意波加载

9. ui_mode.c —— 模式初始化
   - `wave_mode_init()` 初始化波形模式的完整界面：
     清屏 → 绘制上通道矩形 → 绘制波形预览 → 绘制全部菜单（默认未选中）→ 绘制下通道矩形 → 显示当前通道参数 → 显示波形类型
   - 作为其他模式的"基础界面"，被 sync/storage 等模式复用

10. ui_scan.c —— 扫描模式界面
    - `sweep_mode_init()` 初始化扫描模式的完整界面
    - 显示扫描参数：扫描对象（频率/幅度/偏置/占空比）、起始值、截止值、时间、模式（线性/对数）、方向（正向/反向/往返）、状态（开始/停止）
    - `sweep_all_static()` 初始化所有扫描界面的静态元素
    - 处理扫描对象/模式/方向的切换函数：`add_sweep_parameter()` / `add_sweep_mode()` / `add_sweep_dir()`
    - `start_or_end()` 切换起始/截止标签显示
    - `enable_sweep()` 启动/停止扫描

11. ui_select.c —— 数值选择与显示
    - `select_set()` 绘制通道参数数值的每一位数字（选中状态用白字红底高亮当前编辑位，未选中用通道色显示）
    - `drawUnselected()` 绘制下方未选中通道的参数数值
    - `sweep_slected_set()` 绘制扫描参数的数值（起始值/截止值/时间），并附单位（Hz/mv/%）
    - `refresh_selected_item()` 刷新当前选中项的高亮状态，取消前一个高亮
    - `sweep_index_handle()` 处理扫描菜单的索引选择（对象/起始截止/时间/模式/方向）
    - `sweep_ec11_handle()` 处理扫描模式下的旋钮调节（调节起止数值或切换波形类型）
    - `Refresh_Start_Value()` 扫描进行时用红色显示当前实时值

12. ui_storage.c —— 存储/加载界面
    - 波形参数存储与加载的 UI 界面
    - 分页管理：每页 4 个存储位（存01~存04），共 5 页（20 个存储位置）
    - `save_menu_set()` 显示"存XX"或"载XX"标签
    - `save_set_refresh()` 执行实际存储/加载操作，第5项为"更多"翻页
    - `save_mode_init()` 初始化存储/加载模式（基于波形模式界面 + 顶部标题）

13. ui_sync.c —— 同步功能界面
    - 实现双通道参数同步复制
    - `sync_menu_set_refresh()` 切换同步选项（波形/频率/幅度/偏置/占空可独立勾选）
    - `sync_copy()` 当参数被修改时，根据同步标志自动将当前通道的参数复制到另一通道
    - `sync_mode_init()` 初始化同步模式界面（基于波形模式 + 同步菜单 + 顶部"同步"标题）

14. ui_system.c —— 系统信息界面
    - `sys_mode_init()` 显示系统信息：
      型号: SG-28 / 版本: V1.0.2 / 语言: 中文 / 蜂鸣器状态
    - 提供进入存储、加载、同步模式的菜单入口
    - `sys_index_handle()` 处理系统菜单选择：
      INDEX_0 → 存储模式 / INDEX_1 → 加载模式 / INDEX_2 → 同步模式
    - `beep_onoff()` 显示蜂鸣器开关状态

15. ui_waveform.c —— 波形绘制
    - `draw_line_wave()` 逐点连线绘制波形曲线
    - `DrawWaveformDisp()` 初始绘制双通道波形 + 坐标轴（竖轴+横轴）
    - `DrawWaveformDispUpdate()` 刷新波形显示：先擦除（黑色覆盖），再重新生成并绘制
    - 支持标准波形（数学生成）和任意波形（从USB加载的数组数据）两种显示方式
    - 通道1=黄色波形，通道2=青色波形
