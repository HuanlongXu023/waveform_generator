# STM32F407ZE 波形发生器项目技术文档

## 1. 项目概述

本项目是基于 STM32F407ZE 微控制器开发的双通道波形发生器，具有丰富的波形类型、参数调节、扫描功能和测量功能。项目采用 FreeRTOS 实时操作系统，实现了多任务并行处理，确保系统响应及时、稳定可靠。

### 主要功能特点：
- **双通道输出**：支持同时生成两路独立波形
- **丰富的波形类型**：内置 30+ 种标准波形，支持自定义波形
- **参数调节**：可调节频率、幅度、偏置、占空比、相位等参数
- **扫描功能**：支持线性、对数、往返扫描模式
- **测量功能**：内置频率测量功能
- **USB 通信**：支持通过 USB 进行数据传输和控制
- **直观的 LCD 界面**：提供友好的用户操作界面

## 3. 系统架构与模块组成

### 3.1 系统架构
 
本项目采用分层架构设计，从上到下依次为：

1. **应用层**：用户界面、参数控制、功能管理
2. **业务逻辑层**：波形生成、扫描控制、测量功能
3. **硬件抽象层**：STM32 HAL 库、外设驱动
4. **硬件层**：STM32F407ZE 微控制器及其外设

### 3.2 核心模块

#### 3.2.1 波形生成模块

波形生成模块是项目的核心，负责生成各种类型的波形并通过 DAC 输出。

**主要功能**：
- 支持 30+ 种标准波形类型
- 双路独立波形输出
- 精确的参数控制（频率、幅度、偏置、占空比、相位）
- 支持自定义波形

**关键函数**：
- `GenerateWaveform()`：生成指定参数的波形数据
- `StartWaveform()`：启动波形输出
- `SetFrequencyDuty()`：设置频率和占空比

#### 3.2.2 扫描功能模块

扫描功能模块实现了波形参数的自动扫描变化，支持多种扫描模式。

**主要功能**：
- 支持线性、对数扫描模式
- 支持正向、反向、往返扫描方向
- 可扫描频率、幅度、偏置、占空比等参数
- 可设置扫描范围和持续时间

**关键函数**：
- `Waveform_ScanInit()`：初始化扫描功能
- `Waveform_ScanSetParams()`：设置扫描参数
- `Waveform_ScanStart()`：启动扫描
- `Waveform_ScanUpdate()`：更新扫描状态

#### 3.2.3 测量功能模块

测量功能模块实现了频率测量功能，可测量外部信号的频率。

**主要功能**：
- 频率测量
- 计数功能
- 可设置测量闸门时间

**关键函数**：
- `TIM_Freq_Get()`：获取频率测量值

#### 3.2.4 用户界面模块

用户界面模块负责与用户交互，显示波形参数和系统状态，接收用户输入。

**主要功能**：
- 波形参数显示
- 菜单操作
- 波形可视化显示
- 扫描参数设置

**关键函数**：
- `ui_init()`：初始化用户界面
- `ui_task()`：用户界面任务
- `DrawWaveformDisp()`：绘制波形显示

#### 3.2.5 任务管理模块

任务管理模块基于 FreeRTOS 实现，负责系统各任务的调度和管理。

**主要任务**：
- `defaultTask`：默认任务，初始化 USB 设备
- `btuTask`：波形扫描和按键处理
- `UITask`：用户界面更新
- `DebugTask`：调试任务
- `host_deviceTask`：USB 主机设备任务
- `FreqTask`：频率测量任务

## 4. 核心功能详解

### 4.1 波形生成

波形生成是本项目的核心功能，支持多种波形类型，包括：

- **基础波形**：正弦波、矩形波、CMOS方波、脉冲波、直流电平
- **锯齿/三角波**：三角波、上升锯齿波、下降锯齿波
- **阶梯波**：阶梯三角波、正阶梯波、反阶梯波
- **指数波**：正指数波、反指数波、正下降指数、反下降指数
- **对数波**：正对数波、反对数波、正下降对数、反下降对数
- **整流波**：正全波整流、负全波整流、正半波整流、负半波整流
- **特殊脉冲**：洛伦兹脉冲、多音波、均匀白噪声、心电图模拟、梯形波、Sinc脉冲、窄三脉冲、高斯白噪声
- **调制波形**：调幅、调频、线性调频
- **多脉冲**：四脉冲序列、八脉冲序列
- **自定义波形**：支持用户自定义波形

波形参数通过 `ChannelParams_t` 结构体进行管理，包括频率、幅度、偏置、占空比、相位和波形类型。

### 4.2 扫描功能

扫描功能允许波形参数在设定范围内自动变化，支持以下扫描模式：

- **扫描模式**：线性扫描、对数扫描
- **扫描方向**：正向扫描、反向扫描、往返扫描
- **扫描参数**：频率、幅度、偏置、占空比

扫描参数通过 `ScanParams_t` 结构体进行管理，包括扫描使能标志、扫描模式、扫描方向、扫描参数对象、扫描起始值、扫描结束值、扫描总时长等。

### 4.3 测量功能

测量功能主要用于测量外部信号的频率，支持以下特性：

- **频率测量**：测量外部信号的频率
- **计数功能**：对外部信号进行计数
- **闸门时间设置**：可设置测量闸门时间（1s、10s、100s）

### 4.4 USB 通信

USB 通信功能允许设备与计算机进行数据传输和控制，支持以下功能：

- **数据传输**：传输波形数据和测量数据
- **设备控制**：通过 USB 控制设备参数
- **固件升级**：支持通过 USB 进行固件升级

## 6. 使用指南

### 6.1 系统启动

系统启动后，会自动初始化各外设和任务，进入默认的波形模式。默认情况下，通道1和通道2均输出正弦波，频率为1kHz，幅度为1650mV，偏置为1650mV，占空比为50%。

### 6.2 操作流程

1. **选择通道**：通过按键选择要操作的通道（CH1或CH2）
2. **选择波形类型**：通过旋转编码器选择所需的波形类型
3. **调节参数**：通过菜单选择要调节的参数（频率、幅度、偏置、占空比、相位），然后通过旋转编码器调节数值
4. **启动扫描**：进入扫描模式，设置扫描参数，然后启动扫描
5. **测量频率**：进入测量模式，连接外部信号到测量输入端口，系统会自动测量并显示频率

### 6.3 菜单操作

| 菜单选项 | 功能 | 操作方法 |
|---------|------|---------|
| 频率 | 调节波形频率 | 选择后通过旋转编码器调节 |
| 幅度 | 调节波形幅度 | 选择后通过旋转编码器调节 |
| 偏置 | 调节波形偏置 | 选择后通过旋转编码器调节 |
| 占空 | 调节波形占空比 | 选择后通过旋转编码器调节 |
| 相位 | 调节波形相位 | 选择后通过旋转编码器调节 |
| 扫描 | 进入扫描模式 | 按确认键进入，设置扫描参数 |
| 测量 | 进入测量模式 | 按确认键进入，显示频率测量结果 |
| 系统 | 进入系统设置 | 按确认键进入，进行系统相关设置 |

## 7. 注意事项

### 7.1 硬件注意事项

1. **电源要求**：系统需要稳定的 5V 电源输入
2. **信号输出**：输出信号幅度最大为 3.3V，请勿连接超过 3.3V 的外部设备
3. **测量输入**：测量输入信号幅度应限制在 0-3.3V 范围内，避免损坏输入电路
4. **LCD 连接**：确保 LCD 连接正确，避免引脚接反

### 7.2 软件注意事项

1. **参数范围**：调节参数时请注意合理范围，避免设置过大或过小的值
2. **扫描设置**：设置扫描参数时，确保起始值和结束值的合理性，避免扫描范围过大导致系统不稳定
3. **自定义波形**：上传自定义波形时，确保数据格式正确，避免系统崩溃
4. **USB 通信**：使用 USB 通信时，确保驱动安装正确，避免通信失败

### 7.3 维护与故障排除

1. **系统无响应**：检查电源连接，尝试复位系统
2. **波形输出异常**：检查 DAC 输出电路，确认参数设置合理
3. **测量结果不准确**：检查测量输入信号是否符合要求，确保信号稳定
4. **USB 通信失败**：检查 USB 驱动安装情况，尝试重新连接设备
